# 📋 채팅 프로그램

## 📝 프로젝트 개요
이 프로젝트는 프로그램을 구현하는 걸 목표로 합니다.

이름을 설정해서 채팅방에 입장할 수 있고 채팅방도 생성할 수 있습니다.

이 프로젝트는 외부 라이브러리를 사용하지 않고 Built-in 라이브러리만 사용하여 TCP-Socket 연결을 통해 접속을 합니다.

## 📺 화면
  + **이름 입력**
    
  + **채팅방 이름 입력**

  + **채팅방 리스트 출력**
  
  + **채팅방 입장 후 채팅**
  
  + **종료**
